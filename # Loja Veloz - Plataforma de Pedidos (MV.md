# Loja Veloz - Plataforma de Pedidos (MVP Cloud DevOps)

Este repositório contém a infraestrutura como código (IaC), manifests de orquestração do Kubernetes, configurações de ambiente local com Docker Compose e o pipeline de CI/CD automatizado para a modernização da plataforma transacional da **Loja Veloz**, em conformidade com as diretrizes do projeto acadêmico da **UniFECAF**.

---
